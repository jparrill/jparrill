### Key Findings
- **[CNTRLPLANE-3167](https://redhat.atlassian.net/browse/CNTRLPLANE-3167)**: Azure restore fails — downloaded snapshot content is an HTML error page, not etcd data. Root cause: Azure Blob Storage requires presigned URLs for authenticated download; plain `curl` gets auth error response.
- **[OCPBUGS-85084](https://redhat.atlassian.net/browse/OCPBUGS-85084)**: `GITHUB_TOKEN` pushes within a workflow do not trigger further `pull_request`/`push` workflows — known GitHub limitation to prevent infinite loops. Fix requires GitHub App token or API re-trigger.
- **[#8427](https://github.com/openshift/hypershift/pull/8427)** (CNTRLPLANE-3383): `HostedClusterDeleting` condition PR introduces an undocumented NodePool blocking gate in `delete()` that goes beyond pure observability. Namespace deletion enrichment logic has zero test coverage.
- **[#7590](https://github.com/openshift/hypershift/pull/7590)** (CNTRLPLANE-2207): CAPI 1.11 — `json.Unmarshal` failure in conversion-data path silently returns `true`, potentially marking rolling updates complete prematurely. Dead code `GetBackwardCompatibleCAPIImage` still exists.
- **[#8439](https://github.com/openshift/hypershift/pull/8439)** (OCPBUGS-77856): OpenStack classified as "non-cloud" is debatable — OpenStack has Octavia LB support; switching to NodePort could regress existing deployments. `strategy.Route.Hostname` accessed without nil guard is a potential panic.
- **[#8458](https://github.com/openshift/hypershift/pull/8458)** (OCPBUGS-85243): AWS LB scheme annotation fix correct for EKS (ALBC defaults to internal), but KAS public LoadBalancer service has the same gap — follow-up needed.
- **[CNTRLPLANE-3425](https://redhat.atlassian.net/browse/CNTRLPLANE-3425)**: Adopted abandoned PR from departed team member; adds `--limit-crd-install=Azure|GCP` and `--enable-conversion-webhook=false` to Azure case. Pending rehearsals for AKS and GKE jobs.
